Простая консольная версия игры шахматы.
В данной работе упор сделан на механику(обработку команд, 
координат, правильности хода и т.д.)
Реализованы все необходимые правила для игры, условия победы и ничьи и т.д.
К каждой функции прилагается комментарий, писал код так, 
чтобы он был максимально читабельным.

Board.cpp хранит массив символом которые выступают низкоуровневой 
абстракией игровой доски, а также минимальный необходимий интерфейс.

Piece.cpp содержит реализацию абстрактного класа Entity, который
служит базовым для конктретных шахматных фигур. Также здесь хранится 
вектор с объектами, для удобного их использования.

Command.cpp хранит команды для игры, и механизмы считывания этих команд.
